1. Grundlagen

Dieses Kapitel liefert die Basis für das Verständnis von Schwingungen, Wellen und Schall. Zentrale Begriffe, mathematische Grundlagen und physikalische Größen werden hier eingeführt.

1.1 Grundbegriffe

  • Schwingung: Zeitlich periodische Änderung einer physikalischen Größe um einen Gleichgewichtswert, z. B. die Bewegung einer Feder-Masse.
  • Welle: Räumlich und zeitlich periodische Ausbreitung einer Schwingung in einem Medium oder im Raum, z. B. Wasserwellen oder Schall.
  • Schall: Longitudinalwelle in Gasen, Flüssigkeiten oder Festkörpern im Frequenzbereich des menschlichen Hörens (ca. 16 Hz – 20 kHz).
Beispiel: Ein schwingender Körper (z. B. Lautsprechermembran) erzeugt Druckschwankungen in der Luft → Schallwellen.

1.2 Mathematische Grundlagen

Schwingungen und Wellen werden durch Differentialgleichungen beschrieben. Die einfachste Form ist die harmonische Schwingung, die typischerweise an einem Feder-Masse-System betrachtet wird.

Herleitung der Bewegungsgleichung

Ausgangspunkt ist das Feder-Masse-System: Eine Masse m ist an einer Feder befestigt. Nach dem Hookeschen Gesetz gilt:

Rückstellkraft: FFeder = −k · x

  • k: Federkonstante [N/m]
  • x: Auslenkung aus der Ruhelage [m]

Nach dem 2. Newtonschen Gesetz:

Fgesamt = m · ẍ

Setzt man die Federkraft ein:

m · ẍ = −k · x

Umstellen ergibt die Differentialgleichung:

ẍ + (k/m) · x = 0

Definiere die Eigenkreisfrequenz:

ω₀² = k / m

Endgültige Bewegungsgleichung:

ẍ + ω₀² · x = 0

Lösung der Differentialgleichung

Die allgemeine Lösung ist eine harmonische Funktion:

x(t) = A · cos(ω₀·t + φ)

  • A: Amplitude [m]
  • ω₀: Eigenkreisfrequenz [rad/s]
  • φ: Phasenwinkel [rad]

Diese Lösung beschreibt eine zeitlich periodische Bewegung um die Ruhelage.

Warum wichtig?
Die Bewegungsgleichung bildet die Grundlage für alle Schwingungs- und Wellenvorgänge. Ohne sie lassen sich Resonanz, Dämpfung oder erzwungene Schwingungen nicht verstehen.

1.3 Physikalische Größen und SI-Einheiten

  • Amplitude \(A\): Maximale Auslenkung [m]
  • Frequenz \(f\): Schwingungen pro Sekunde [Hz] = [1/s]
  • Kreisfrequenz \(\omega\): \[ \omega = 2\pi f \quad [\text{rad/s}] \]
  • Periode \(T\): Dauer einer Schwingung: \[ T = \frac{1}{f} \]
  • Wellenlänge \(\lambda\): Abstand zweier benachbarter Maxima [m]
  • Ausbreitungsgeschwindigkeit \(c\): \[ c = f \cdot \lambda \]
Warum wichtig?
Ohne die Grundlagen zu Schwingungen und Wellen können Resonanzphänomene, Schallausbreitung oder akustische Systeme nicht verstanden werden. Dieses Kapitel ist die Basis für alle weiteren Themen.

2. Schwingungen

In diesem Kapitel betrachten wir verschiedene Arten von Schwingungen: freie, gedämpfte und erzwungene Schwingungen. Jede dieser Bewegungen wird durch eine spezifische Differentialgleichung beschrieben.

2.1 Freie harmonische Schwingung

Bei der freien Schwingung wirkt nur die Rückstellkraft der Feder:

Gleichung: ẍ + ω₀² · x = 0

Lösung:

x(t) = A · cos(ω₀·t + φ)

  • A: Amplitude [m]
  • ω₀: Eigenkreisfrequenz [rad/s]
  • φ: Anfangsphase [rad]

Eigenschaften:

  • Periodendauer: T = 2π / ω₀
  • Frequenz: f = ω₀ / (2π)
  • Energie bleibt konstant (keine Dämpfung)

2.2 Gedämpfte Schwingung

Bei realen Systemen treten Reibung oder Luftwiderstand auf → Dämpfungskraft proportional zur Geschwindigkeit:

Dämpfungskraft: Fd = −b · ẋ

Bewegungsgleichung:

m · ẍ + b · ẋ + k · x = 0

Durch Division durch m:

ẍ + (b/m) · ẋ + (k/m) · x = 0

Setze:

  • 2δ = b/m (Dämpfungsmaß)
  • ω₀² = k/m (ungedämpfte Eigenfrequenz)

ẍ + 2δ · ẋ + ω₀² · x = 0

Lösung hängt von δ und ω₀ ab:

  • Schwach gedämpft (δ < ω₀): Schwingung mit abklingender Amplitude
  • Kritisch gedämpft (δ = ω₀): schnellste Rückkehr ohne Schwingen
  • Stark gedämpft (δ > ω₀): kein Schwingen, langsames Zurückgehen

Lösung für δ < ω₀:

x(t) = A · e−δt · cos(ωd · t + φ)

  • ωd = √(ω₀² − δ²) → gedämpfte Eigenfrequenz

2.3 Erzwungene Schwingung

Wird das System durch eine periodische äußere Kraft F(t) = F₀·cos(Ω·t) angeregt, lautet die Bewegungsgleichung:

ẍ + 2δ · ẋ + ω₀² · x = (F₀/m) · cos(Ω·t)

Allgemeine Lösung = Überlagerung von:

  • Homogene Lösung: Gedämpfte Schwingung (wie oben)
  • Partikuläre Lösung: Stationäre Schwingung mit Frequenz Ω

Stationäre Lösung:

x(t) = X · cos(Ω·t − φ)

Amplitude X abhängig von Ω:

X = (F₀/m) / √((ω₀² − Ω²)² + (2δΩ)²)

Resonanz: Bei Ω ≈ ω₀ erreicht die Amplitude ein Maximum → gefährlich für Bauwerke (Brücken, Maschinen).

2.4 Energiebetrachtung

Für eine ungedämpfte Schwingung gilt:

  • Kinetische Energie: Ekin = ½ m ẋ²
  • Potentielle Energie: Epot = ½ k x²
  • Gesamtenergie: E = konstant

Bei Dämpfung: E nimmt exponentiell ab → Energieverlust durch Reibung/Wärme.

2.5 Reale Anwendung: Schwingungen im Auto (Stoßdämpfer)

Ein klassisches Beispiel für gedämpfte Schwingungen ist das Feder-Dämpfer-System im Auto. Die Fahrzeugkarosserie wird durch Federn und Stoßdämpfer vor Schwingungen nach einem Schlagloch geschützt.

Modellannahmen:

  • Masse des Autos (pro Rad): m = 250 kg
  • Federkonstante: k = 15.000 N/m
  • Dämpfungskonstante: b = 1.000 Ns/m

Bewegungsgleichung:

m · ẍ + b · ẋ + k · x = 0

Berechnung der Parameter:

  • Eigenkreisfrequenz: ω₀ = √(k / m) = √(15.000 / 250) = √60 ≈ 7,75 rad/s
  • Frequenz: f₀ = ω₀ / (2π) ≈ 7,75 / 6,283 ≈ 1,23 Hz
  • Dämpfungsmaß: δ = b / (2m) = 1.000 / (2·250) = 2 s⁻¹

Prüfe Dämpfung:

  • δ < ω₀ → System ist unterkritisch gedämpft → das Auto schwingt nach dem Schlagloch noch etwas, aber die Amplitude nimmt schnell ab.
Interpretation:
Ohne Dämpfer würde das Auto mit etwa 1,2 Hz dauerhaft schwingen – sehr unangenehm für die Insassen! Dank Dämpfung klingt die Schwingung nach wenigen Sekunden ab.

3. Wellen

Wellen sind Schwingungen, die sich im Raum ausbreiten. In diesem Kapitel betrachten wir mechanische Wellen, ihre mathematische Beschreibung und Eigenschaften.

3.1 Wellenarten

Wellen können nach der Ausbreitungsrichtung und der Schwingungsrichtung unterschieden werden:

  • Transversalwellen: Schwingung senkrecht zur Ausbreitungsrichtung (z. B. Wasserwellen, Seilwellen).
  • Longitudinalwellen: Schwingung in Ausbreitungsrichtung (z. B. Schallwellen in Luft).

Beispiele:

  • Seilwelle → transversal
  • Schallwelle → longitudinal

3.2 Wellengleichung

Ausgangspunkt: Eine Welle ist eine zeit- und ortsabhängige Auslenkung y(x, t). Für eine ideale, verlustfreie Welle gilt:

Partielle Differentialgleichung:

∂²y/∂t² = c² · ∂²y/∂x²

Herleitung: Newtons Gesetz für ein elastisches Medium → Zusammenhang zwischen Trägheit und elastischer Rückstellung.

Allgemeine Lösung:

y(x, t) = f(x − ct) + g(x + ct)

  • f: Welle nach rechts
  • g: Welle nach links

3.3 Harmonische Welle

Für eine harmonische Ausbreitung:

y(x, t) = A · cos(kx − ωt + φ)

  • A: Amplitude
  • k = 2π/λ: Wellenzahl
  • ω = 2πf: Kreisfrequenz

Zusammenhang:

c = λ · f = ω / k

3.4 Energieübertragung

Wellen transportieren Energie, ohne dass sich Materie mit der Welle fortbewegt (nur Schwingungen um eine Ruhelage).

Leistung ∝ A² · f² (abhängig von Amplitude und Frequenz)

3.5 Überlagerung und stehende Wellen

Treffen zwei Wellen gleicher Frequenz aufeinander, überlagern sie sich (Superposition):

  • Konstruktive Interferenz: Amplituden addieren sich.
  • Destruktive Interferenz: Amplituden heben sich auf.

Stehende Wellen entstehen, wenn sich zwei gleich starke Wellen entgegengesetzt ausbreiten:

y(x, t) = 2A · sin(kx) · cos(ωt)

Knoten: Stellen ohne Auslenkung → Abstand = λ/2.

3.6 Reflexion, Brechung, Beugung

  • Reflexion: Wellen werden an Grenzflächen zurückgeworfen.
  • Brechung: Änderung der Ausbreitungsrichtung bei Geschwindigkeitswechsel → Snelliussches Gesetz.
  • Beugung: Wellen breiten sich auch in Schattenräumen aus → charakteristisch für Schall und Licht.
